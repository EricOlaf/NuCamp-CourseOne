Visual Studio Code Live Server Extension

Installation

The first step is to install the live server extension for VS code.

  1. In VS code, click on the Extension icon in the upper left corner
  2. In the search box write: Live Server
  3. You should see the extension on the right: click install

Using Live Server

  • Open a project and click to Go Live from the status fbar to turn the server on and off.
  • Right click on a HTML file from the Explorer window and click, Open with Live Server.
